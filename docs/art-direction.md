# Pixel Character Art Direction

## Goal
- Build a cohesive 6-character chibi pixel-art set for an AI agent office UI.
- Cute, warm, cozy, premium, and readable in a professional dashboard.

## Core Rules
- Canvas: 64x64 per character.
- Chibi proportions: large head, small body, short limbs.
- Thick dark outline (#2a1c16).
- Top-left lighting.
- No anti-aliasing or blur.
- Limited palette: 12–16 colors per character.
- One primary color family + one accent color + one role prop.

## Shared Anatomy
- Head: 24x24 block with simple face.
- Body: 20x18 block with compact torso.
- Legs: short with small feet.
- Arms: minimal, attached to torso.

## Vibe
- Office-tech agent style, not fantasy RPG.
- Slightly game-like, but suitable for modern dashboards.
- Expressive, minimal faces.

## Export Requirements
- Individual PNGs per role.
- Combined sprite sheet.
- Palette JSONs per role.
